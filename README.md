# 🤖 Template de Agente para WhatsApp (Node.js)

Um boilerplate limpo, simples e direto ao ponto para você criar o seu próprio bot de WhatsApp. Ideal para automatizar atendimentos e integrar com inteligências artificiais.

## ✨ Destaques
- **Sessão Salva Localmente:** Faça o login via QR Code apenas uma vez.
- **Leve e Rápido:** Utiliza a biblioteca oficial `whatsapp-web.js`.
- **Pronto para Escalar:** Estrutura básica perfeita para plugar APIs como OpenAI (ChatGPT) ou Gemini.

## 🚀 Como rodar na sua máquina

1. Instale as dependências:
\`\`\`bash
npm install
\`\`\`
2. Inicie o bot:
\`\`\`bash
node index.js
\`\`\`
3. Escaneie o QR Code que aparecerá no terminal com o seu WhatsApp.
4. Envie a mensagem `ping` para testar!
